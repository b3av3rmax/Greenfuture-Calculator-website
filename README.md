### 탄소 발자국 계산기 소개

일상 속 플라스틱 소비량을 측정하고, 그에 따른 환경 영향을 시각적으로 확인할 수 있도록 개발된 웹 기반 환경 교육 플랫폼이다.

음식·주방, 욕실, 생활용품 등 다양한 생활 영역에서 사용하는 플라스틱 제품을 입력하여 연간 플라스틱 사용량과 평생 플라스틱 소비량을 계산할 수 있으며, 이를 탄소배출량 및 환경 영향 지표로 변환하여 확인할 수 있습니다.

복잡한 환경 데이터를 누구나 쉽게 이해할 수 있는 형태로 제공하여 지속가능한 소비 습관 형성과 환경 인식 개선을 목표로 개발되었음.

### 주요 기능

## 플라스틱 사용량 계산

* 생활 속 플라스틱 제품 사용량 입력
* 품목별 플라스틱 무게 자동 계산
* 연간 플라스틱 소비량 산출
* 평생 플라스틱 소비량 예측

## 환경 영향 분석

* 플라스틱 사용량 기반 탄소배출량 환산
* 환경 영향 지표 제공
* 나무 식재 환산 정보 제공
* 친환경 실천 효과 시뮬레이션

## 데이터 시각화

* 카테고리별 사용량 차트
* 플라스틱 소비 비율 분석
* 결과 그래프 및 통계 시각화
* 직관적인 환경 영향 대시보드

## 사용자 경험(UX)

* 반응형 웹 디자인
* 모바일 및 데스크톱 지원
* 직관적인 입력 인터페이스
* 빠른 실시간 계산 결과 제공

## Front-End

* HTML5
* Tailwind CSS
* JavaScript (ES6+)

## Data Visualization

* Chart.js

## Deployment

* GitHub Pages
* Netlify

## 프로젝트 목표

플라스틱 소비는 일상에서 쉽게 체감하기 어렵지만 장기적으로는 상당한 환경 부담을 유발합니다. 본 프로젝트는 사용자가 자신의 플라스틱 소비 패턴을 수치로 확인하고, 환경에 미치는 영향을 이해할 수 있도록 지원하는 환경 교육 도구로 제작되었습니다.

이를 통해 개인의 친환경 실천을 유도하고 지속가능한 생활문화 확산에 기여하는 것을 목표로 합니다.

## 기대 효과
